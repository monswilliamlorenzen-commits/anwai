# anwai — Next.js

Modern, ultra-responsiv webbplats för anwai byggd med Next.js 15, TypeScript och Tailwind CSS.

## Funktioner

- **Next.js 15** med App Router
- **Svenska & Engelska** — språkväxling via `/sv` och `/en`
- **Responsiv design** — optimerad för mobil, surfplatta och desktop
- **Tailwind CSS** — snabb styling med utility classes
- **Statisk generering** — snabba sidladdningar

## Kom igång

```bash
npm install
npm run dev
```

Öppna [http://localhost:3000](http://localhost:3000) — du omdirigeras automatiskt till `/sv`.

## Sidor

- `/sv` eller `/en` — startsida
- `/sv/about`, `/en/about` — Om oss
- `/sv/discography`, `/en/discography` — Diskografi
- `/sv/merch`, `/en/merch` — Merch
- `/sv/contact`, `/en/contact` — Kontakt

## Bygg för produktion

```bash
npm run build
npm start
```
